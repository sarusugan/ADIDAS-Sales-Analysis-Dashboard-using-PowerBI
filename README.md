**ADIDAS-Sales-Analysis-Dashboard-using-PowerBI**
This project presents an interactive sales analytics dashboard built using Microsoft Power BI to analyze Adidas sales performance across regions, products, and time periods.
The dashboard helps stakeholders make data-driven decisions by visualizing key sales metrics and trends.
**Project Overview**
The Adidas Sales Analysis Dashboard provides a comprehensive view of sales data, enabling users to:
Monitor overall sales performance
Identify top-performing products and regions
Analyze revenue trends over time
Understand customer purchasing behavior
This dashboard is designed for business intelligence, reporting, and strategic insights.
 **Dashboard Features**
 Total Sales & Revenue Analysis
 Region-wise Sales Performance
 Product Category Analysis
 Time-based Trends (Monthly / Yearly)
 Key Performance Indicators (KPIs)
 Interactive Filters & Slicers
** Tools & Technologies Used**
Microsoft Power BI
Power Query (Data Cleaning & Transformation)
DAX (Calculated Measures & KPIs)
Excel / CSV Dataset (as data source)
